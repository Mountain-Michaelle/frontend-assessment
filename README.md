Readme for frontend assessment project


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.



**Technical & Functional Overview**
Framework: Next.js (React)
Language: TypeScript
Responsive Design: Optimized for desktop and mobile
Component Structure: Small, clean, and reusable components
Form Handling: Controlled inputs with local state
Data Handling: No backend integration; mock data used as required

**Optional Enhancements Implemented**
Reusable UI components
Basic client-side form validation
Simple loading and error states
Thoughtful and scalable folder structure

Local state is used instead of global state management due to project size.

Fonts/assets are handled with build reliability in mind.

**Notes**
The project prioritizes clarity, maintainability, and close alignment with the provided design and requirements.
